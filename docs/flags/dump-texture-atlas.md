# Dump Texture Atlas

**Property:** `MC_DEBUG_DUMP_TEXTURE_ATLAS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Writes all texture atlases to disk as PNG files when the game loads or reloads resource packs. The files are saved to the game directory and show the full packed atlas layout, including every sprite and its UV coordinates within the sheet.

## When to use

Enable when debugging texture packing issues, UV misalignment, or verifying that a custom texture appears at the expected position in the atlas.
