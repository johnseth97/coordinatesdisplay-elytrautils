# Structure Edit Mode

**Property:** `MC_DEBUG_STRUCTURE_EDIT_MODE`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Structure blocks placed during generation are preserved in the world instead of being consumed or replaced. This exposes the internal structure block layout used during assembly, allowing inspection and manual editing of generated structures in-place.

## When to use

Enable when iterating on structure templates and needing to see exactly which structure blocks were placed and where during the generation pass.
