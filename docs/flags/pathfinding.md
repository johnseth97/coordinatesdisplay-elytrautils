# Pathfinding

**Property:** `MC_DEBUG_PATHFINDING`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Draws the active pathfinding routes of all nearby mobs as colored lines connecting navigation nodes in the world. Each mob's current path is rendered in real time, with color variations indicating path segments and their traversal cost. This makes it straightforward to see exactly which route a mob has chosen and where it may be getting stuck on terrain or obstacles.

## When to use

Enable this when debugging mob navigation issues, such as mobs unable to reach targets, taking unexpected routes, or getting stuck near blocks and ledges.
