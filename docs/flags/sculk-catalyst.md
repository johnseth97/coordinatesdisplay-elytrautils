# Sculk Catalyst

**Property:** `MC_DEBUG_SCULK_CATALYST`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Visualizes the sculk spread radius around sculk catalysts, shows the accumulation of charge from mob deaths, and highlights bloom event triggers as sculk is placed. The spread radius is drawn as a volume around the catalyst, and bloom events — which determine where sculk features grow — are indicated as they fire. Charge values are displayed numerically as they accumulate.

## When to use

Use this when testing sculk spread mechanics, tuning bloom event thresholds, or debugging catalysts that are not spreading sculk at the expected rate or range.
