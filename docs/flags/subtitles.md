# Subtitles

**Property:** `MC_DEBUG_SUBTITLES`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Forces all sound events to display subtitle entries in the subtitle overlay, including sounds that normally suppress their subtitle. This exposes every sound being played in the game, along with its resource location, making it easy to identify which sound event corresponds to a given in-game audio cue.

## When to use

Enable when mapping sound events for resource packs, datapacks, or mods, or when verifying that specific sounds trigger at the correct moments.
