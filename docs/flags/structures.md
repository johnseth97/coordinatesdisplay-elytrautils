# Structures

**Property:** `MC_DEBUG_STRUCTURES`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Draws axis-aligned bounding boxes around all loaded structure pieces and their overall start positions in the world. Each structural piece is outlined individually, allowing you to see how a multi-piece structure like a stronghold or mineshaft is assembled from its component parts. Start positions are marked separately from the bounding volumes of individual pieces.

## When to use

Enable this when developing or debugging custom structures, checking piece connectivity, or verifying that structure piece boundaries align correctly with the world.
