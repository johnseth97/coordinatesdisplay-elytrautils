# Monitor Tick Times

**Property:** `MC_DEBUG_MONITOR_TICK_TIMES`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Records per-tick timing data for both the server and client loops. The data is stored internally and can be viewed via the built-in profiler (F3+L). Enables finer-grained tick time breakdowns than the default profiler output provides.

## When to use

Enable when investigating lag spikes or inconsistent tick rates and needing per-tick granularity beyond what the standard TPS display shows.
