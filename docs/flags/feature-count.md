# Feature Count

**Property:** `MC_DEBUG_FEATURE_COUNT`
**Tab:** World Generation
**Requires restart:** Yes

## Description

Shows per-chunk feature placement counts as a heatmap overlay on the terrain, with warmer colors indicating chunks where more features (trees, ores, decorations, structures) were placed during generation. This gives a quick spatial summary of feature density across a generated area. Chunks with anomalously high counts may indicate misconfigured feature frequency or overlapping placement passes.

## When to use

Enable this when auditing feature distribution across biomes, diagnosing overly dense or sparse feature generation, or verifying that custom features place at the intended frequency.
