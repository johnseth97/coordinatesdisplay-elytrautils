# Disable Structures

**Property:** `MC_DEBUG_DISABLE_STRUCTURES`
**Tab:** World Generation
**Requires restart:** Yes

## Description

Prevents all structures — villages, strongholds, mineshafts, temples, shipwrecks, and every other registered structure type — from being placed during world generation. Chunks generate with terrain, features, and decorations intact but no structure starts or pieces are injected.

## When to use

Use when testing terrain or feature generation in isolation, or when profiling chunk generation time with structures as the isolated variable.
