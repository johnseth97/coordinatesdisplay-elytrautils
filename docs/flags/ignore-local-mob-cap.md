# Ignore Local Mob Cap

**Property:** `MC_DEBUG_IGNORE_LOCAL_MOB_CAP`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

The integrated server ignores the global mob spawning cap, allowing mobs to spawn without the usual per-category limits. This affects all spawn categories (monster, creature, ambient, water creature, etc.). Mob counts can grow arbitrarily large, which will impact performance.

## When to use

Enable when testing mob spawning mechanics, biome spawn lists, or mob farm designs where the cap would interfere with measuring natural spawn rates.
