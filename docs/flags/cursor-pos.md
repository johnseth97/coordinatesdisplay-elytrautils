# Cursor Pos

**Property:** `MC_DEBUG_CURSOR_POS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Adds exact cursor pixel coordinates and screen-space position to the F3 overlay. Displays both the raw window pixel position and the scaled GUI coordinate, which differ when the GUI scale setting is not 1:1.

## When to use

Enable when developing or debugging UI screens where precise cursor hit-testing or element positioning needs to be verified.
