# Brain

**Property:** `MC_DEBUG_BRAIN`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Overlays floating text above each entity showing the current values stored in its brain memory modules, including home position, attack target, walk target, look target, and any other active memory entries. The text updates every tick, giving a live view into the entity's internal decision-making state. This is particularly useful for villagers, piglins, and other entities that rely heavily on the brain/memory system.

## When to use

Enable this when debugging entity behavior driven by the brain system, such as villagers not going home, piglins not targeting correctly, or memory values not being set or cleared as expected.
