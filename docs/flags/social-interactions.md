# Social Interactions

**Property:** `MC_DEBUG_SOCIAL_INTERACTIONS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Shows social interaction debug state for players, including mute status and block reasons from the social interactions screen. Debug labels appear above affected players showing their current interaction state as seen by the local client.

## When to use

Enable when testing the social interactions system, player muting, or verifying that block/mute state is correctly synchronized between the client and server.
