# Block Break

**Property:** `MC_DEBUG_BLOCK_BREAK`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Overlays blocks that are currently being mined with floating text showing the break progress as a percentage and the remaining hardness value. This gives precise numerical feedback on mining progress rather than relying solely on the visual crack animation. Both the progress fraction and the underlying hardness are displayed, which is useful for verifying tool efficiency calculations.

## When to use

Use this when testing custom block hardness values, tool enchantment effects on break speed, or verifying that mining speed modifiers apply correctly.
