# Aquifers

**Property:** `MC_DEBUG_AQUIFERS`
**Tab:** World Generation
**Requires restart:** Yes

## Description

Overlays aquifer cell boundaries as colored volumes during world generation, and labels each cell with its assigned fluid level and fluid type (water or lava). Aquifers carve the world into local fluid zones with independent fluid levels, so this overlay reveals the cell structure that determines where underground pools form and at what height. Boundaries between cells with different fluid types are highlighted distinctly.

## When to use

Use this when investigating unexpected underground fluid placement, debugging aquifer noise parameters, or verifying that cell boundaries align with intended geological features.
