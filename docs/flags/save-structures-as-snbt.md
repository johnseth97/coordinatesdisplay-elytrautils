# Save Structures As SNBT

**Property:** `MC_DEBUG_SAVE_STRUCTURES_AS_SNBT`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Structure files are saved as human-readable SNBT (Stringified NBT) text files instead of binary NBT when using the structure block save function. The resulting `.snbt` files can be opened in any text editor and diffed with version control tools, making it easy to review structural changes.

## When to use

Enable when authoring or modifying structure templates and wanting to track changes in version control or review diffs between structure versions.
