# Unlock All Trades

**Property:** `MC_DEBUG_UNLOCK_ALL_TRADES`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

All villager trade tiers are immediately unlocked regardless of trading history or XP level. Normally a villager must be traded with repeatedly to unlock higher tiers; this flag bypasses that progression entirely. All offers from every tier appear in the trading UI from the first interaction.

## When to use

Enable when testing economy balance, loot table outputs, or any mechanic that requires access to high-tier villager trades without grinding.
