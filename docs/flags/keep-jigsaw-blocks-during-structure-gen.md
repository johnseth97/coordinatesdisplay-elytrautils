# Keep Jigsaw Blocks During Structure Gen

**Property:** `MC_DEBUG_KEEP_JIGSAW_BLOCKS_DURING_STRUCTURE_GEN`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Jigsaw blocks are not removed after structure generation completes. Normally the game replaces jigsaw blocks with their final block state once the structure is assembled. With this flag the jigsaw blocks remain in-world, making every connection point and pool assignment visible for inspection.

## When to use

Enable when designing or debugging jigsaw-based structures to verify that pool assignments, joint types, and connection orientations are correct.
