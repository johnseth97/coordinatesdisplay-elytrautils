# Carvers

**Property:** `MC_DEBUG_CARVERS`
**Tab:** World Generation
**Requires restart:** Yes

## Description

Draws the volumetric regions carved out by cave and canyon carvers during world generation, along with the intersection zones where multiple carver volumes overlap. Each carver pass is represented as a translucent volume, making it possible to see the raw carver shapes before surface decoration and feature placement modify the terrain. Intersection regions appear brighter where multiple carvers overlap.

## When to use

Use this when developing custom carver implementations, tuning carver size and frequency parameters, or investigating unexpected cave shapes in generated terrain.
