# Raids

**Property:** `MC_DEBUG_RAIDS`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Displays the current raid wave state, patrol spawn positions, and omen level above the raid center location. Wave progression, remaining raider counts, and the raid's internal status are shown as floating text. Patrol spawn points are marked in the world so you can see where the game intends to spawn incoming waves.

## When to use

Use this when testing raid mechanics, debugging wave progression issues, or verifying that omen levels and spawn positions behave correctly during development.
