# Experimental Redstonewire Update Order

**Property:** `MC_DEBUG_EXPERIMENTAL_REDSTONEWIRE_UPDATE_ORDER`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Highlights the update propagation order of redstone wire using the experimental wire evaluator, coloring wire segments to show the sequence in which power changes are applied. This visualizer is tied specifically to the experimental redstone algorithm and does not reflect the behavior of the standard evaluator. It allows developers to compare update ordering between the two implementations.

## When to use

Use this when evaluating or comparing the experimental redstone wire algorithm's propagation behavior against the legacy implementation.
