# Ore Veins

**Property:** `MC_DEBUG_ORE_VEINS`
**Tab:** World Generation
**Requires restart:** No

## Description

Renders the generation boundaries of ore veins and labels each vein with its type classification (copper-type or iron-type) during world generation. The bounding volume of each vein is drawn so you can see the extent of the deposit before it is carved into the terrain noise. Vein type and orientation are indicated by color and label.

## When to use

Enable this when debugging ore vein placement, tuning vein size and frequency data-driven configs, or verifying that vein types appear at the correct depth ranges.
