# Bypass Realms Version Check

**Property:** `MC_DEBUG_BYPASS_REALMS_VERSION_CHECK`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Skips the version compatibility check when connecting to a Realms server. Normally the client refuses to join a Realm running a different game version. With this flag the connection proceeds regardless of the version mismatch.

## When to use

Enable when testing Realms connectivity from a snapshot build against a Realm running a different version.
