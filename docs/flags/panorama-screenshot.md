# Panorama Screenshot

**Property:** `MC_DEBUG_PANORAMA_SCREENSHOT`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Enables the internal panorama capture mode that takes a 6-sided cubemap screenshot set. The six images are saved to disk in a format suitable for use as a main menu background panorama. The capture rotates the camera to each cube face automatically.

## When to use

Enable when creating a custom main menu background panorama for a modpack or resource pack.
