# Vault Detects Sheep As Players

**Property:** `MC_DEBUG_VAULT_DETECTS_SHEEP_AS_PLAYERS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Vault blocks treat nearby sheep as players when determining whether to dispense rewards and track unlock state. Allows a single player to test vault reward distribution and per-player unlock logic by using sheep as stand-ins for additional players.

## When to use

Enable when testing vault reward tables or verifying that per-player unlock state is tracked correctly without needing multiple accounts.
