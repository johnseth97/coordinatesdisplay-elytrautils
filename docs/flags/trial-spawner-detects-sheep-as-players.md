# Trial Spawner Detects Sheep As Players

**Property:** `MC_DEBUG_TRIAL_SPAWNER_DETECTS_SHEEP_AS_PLAYERS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Trial spawner activation logic counts nearby sheep as players for the purpose of detecting whether the spawner should activate. This allows a single player to test trial spawner behavior — including wave progression and reward dispensing — by placing sheep near the spawner instead of needing multiple players.

## When to use

Enable when testing trial chamber mechanics, spawner wave counts, or reward loot tables without needing multiple player accounts.
