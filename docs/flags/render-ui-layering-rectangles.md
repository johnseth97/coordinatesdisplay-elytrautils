# Render UI Layering Rectangles

**Property:** `MC_DEBUG_RENDER_UI_LAYERING_RECTANGLES`
**Tab:** Misc
**Requires restart:** Yes

## Description

Draws colored rectangles over each GUI layer boundary, visualizing the render order and stacking depth of UI components. Each layer gets a distinct color tint, making overlapping or incorrectly ordered layers immediately visible.

## When to use

Enable when debugging z-ordering issues in custom screens, overlapping widgets, or UI components that render in the wrong layer.
