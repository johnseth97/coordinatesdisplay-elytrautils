# UI Narration

**Property:** `MC_DEBUG_UI_NARRATION`
**Tab:** Misc
**Requires restart:** Yes

## Description

Outputs debug information about the accessibility narration system to the log, including which UI components are currently queued for narration and what text is being passed to the screen reader. Helps verify that all interactive widgets have correct narration messages.

## When to use

Enable when developing accessible UI screens and needing to verify that narration messages are correct, complete, and fire at the right times.
