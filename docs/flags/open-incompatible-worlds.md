# Open Incompatible Worlds

**Property:** `MC_DEBUG_OPEN_INCOMPATIBLE_WORLDS`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

The world selection screen allows loading worlds created in incompatible game versions without showing a warning or triggering a conversion prompt. The world is opened as-is; no data fixer migration is forced. Using this on worlds with significant version gaps may cause chunk corruption or missing data.

## When to use

Enable when you need to open an old world in a newer snapshot to inspect its raw state before deciding whether to migrate it.
