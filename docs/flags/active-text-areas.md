# Active Text Areas

**Property:** `MC_DEBUG_ACTIVE_TEXT_AREAS`
**Tab:** Misc
**Requires restart:** Yes

## Description

Highlights currently focused and active text input fields with a debug outline. Shows which `EditBox` widget has keyboard focus at any given moment, making it easy to verify focus transitions when tabbing between fields or when focus is programmatically set.

## When to use

Enable when developing or debugging UI screens with multiple text inputs to verify that focus management behaves correctly.
