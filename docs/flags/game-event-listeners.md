# Game Event Listeners

**Property:** `MC_DEBUG_GAME_EVENT_LISTENERS`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Overlays vibration listeners such as sculk sensors and wardens with a visual representation of their detection range and the last game event they heard. The range sphere or box is drawn around each listener, and a label indicates the most recently received event type. This helps verify that listeners are responding to the correct event sources at the expected distances.

## When to use

Use this when designing sculk sensor circuits, testing warden aggro ranges, or debugging vibration-based contraptions that are not triggering as expected.
