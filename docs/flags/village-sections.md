# Village Sections

**Property:** `MC_DEBUG_VILLAGE_SECTIONS`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Renders village section boundaries as colored outlines in the world along with per-section counts for POIs, beds, and meeting points. This reveals how the game partitions the world into sections for village tracking and gives a count of each resource type the village system has registered within each section. Sections that are part of an active village appear differently from unaffiliated sections.

## When to use

Enable this when debugging village formation, villager job assignment failures, or issues with beds and meeting points not being recognized by the village system.
