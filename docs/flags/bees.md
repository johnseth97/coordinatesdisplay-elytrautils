# Bees

**Property:** `MC_DEBUG_BEES`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Draws lines from each bee to its owned hive, labels each bee with its current behavioral state (returning, pollinating, entering hive, etc.), and shows hive ownership information above each hive block. This makes it easy to see how bees are distributed across hives and whether they are navigating home correctly. Bees that have lost their hive reference are identifiable by missing path lines.

## When to use

Enable this when debugging bee hive population, bees not returning to their hive, or hive capacity and ownership issues.
