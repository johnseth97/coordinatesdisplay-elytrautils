# Poi

**Property:** `MC_DEBUG_POI`
**Tab:** Renderers
**Requires restart:** Yes

## Description

Renders all nearby points of interest — including beds, bells, job site blocks, and nether portals — with floating labels showing their type and current occupancy or reservation counts. POIs that are fully occupied appear distinctly from those with available slots. This gives a direct view into the data the village system uses to assign villagers to homes and professions.

## When to use

Use this when troubleshooting villager profession assignment, bed linking, or any scenario where POIs are not being recognized or are showing incorrect occupancy.
