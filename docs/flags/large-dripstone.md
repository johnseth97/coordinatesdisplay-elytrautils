# Large Dripstone

**Property:** `MC_DEBUG_LARGE_DRIPSTONE`
**Tab:** World Generation
**Requires restart:** Yes

## Description

Renders the generation boundaries of large dripstone clusters during world generation debugging, showing the volumes within which the game attempts to place dripstone columns and pointed dripstone. These boundaries are only meaningful when examining newly generated chunks and are not visible during normal gameplay. The overlay helps verify that cluster placement parameters produce correctly sized formations.

## When to use

Enable this when tuning large dripstone cluster generation parameters or verifying that dripstone formations are being placed within their intended generation bounds.
