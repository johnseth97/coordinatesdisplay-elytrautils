# Show Server Debug Values

**Property:** `MC_DEBUG_SHOW_SERVER_DEBUG_VALUES`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Exposes additional server-side debug entries in the F3 overlay, such as tick rate variance, entity processing counts, and scheduled tick queues. These values are normally hidden in release builds and only appear in Mojang's internal dev environment.

## When to use

Enable when profiling integrated server performance or investigating tick-rate irregularities that aren't visible through the standard F3 display.
