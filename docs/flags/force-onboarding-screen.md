# Force Onboarding Screen

**Property:** `MC_DEBUG_FORCE_ONBOARDING_SCREEN`
**Tab:** Gameplay
**Requires restart:** Yes

## Description

Forces the new-player onboarding and tutorial screen to appear on every launch regardless of whether the player has already completed or dismissed it. The completion flag stored in options is ignored while this flag is active.

## When to use

Enable when testing the onboarding flow UI or verifying that tutorial screen behavior is correct after changes to the screen's logic or layout.
